# InkTrace

A personal novel reading project focused on a clean reading experience and practical daily use.

InkTrace is the novel reader I actively use. This repository exists to support reading, organizing, and improving the overall reading workflow over time.

## Purpose

- Provide a dedicated place for reading novels
- Keep the reading experience simple and focused
- Support ongoing personal use and iterative improvement

## Project Direction

InkTrace is intended to be a practical reader rather than a demo-only project. The goal is to keep it useful in real daily reading scenarios while continuing to refine the experience.

## Focus Areas

- Reading experience
- Content organization
- Personal usability
- Long-term maintainability

## Current Status

This project is in active personal use.
Features and structure may continue to evolve as the reading workflow improves.

## Notes

This repository is maintained as a real-use personal project.
It may change over time based on reading habits, interface improvements, and future feature ideas.
